# Freelance Hours

**Projeto desenvolvido com a Rocketseat em um curso gratuito de PHP, utilizando Laravel e Livewire**

## Para executar o projeto
Primeiro instale as dependências:
```
composer install
npm install
```

Em seguida, basta rodar o projeto e os estilos
```
php artisan serve
npm run dev
```
